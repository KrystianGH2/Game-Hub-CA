## Getting Started

### Installing

To run the project locally, follow these steps:

1. Clone the repo:

```bash
 git clone https://github.com/KrystianGH2/Game-Hub-CA.git
```

2. Install the dependencies:

```bash
cd Game-Hub-CA
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
